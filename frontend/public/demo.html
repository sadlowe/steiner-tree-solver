Relier 3 maisons avec le moins de route possible

Où placer le carrefour pour minimiser la longueur totale du réseau ?

🏘️
Le problème :
Un village veut construire des routes pour relier 3 maisons entre elles.

L'objectif : dépenser le moins possible, c'est-à-dire construire le moins de route possible.

La solution : ajouter un carrefour au bon endroit pour partager le chemin commun !
Comment utiliser la démo
1
Cliquez sur la carte pour placer 3 maisons
2
La route directe (orange) s'affiche — comme si on reliait chaque maison sans carrefour
3
Cliquez sur « Trouver le carrefour » pour voir la route optimale avec carrefour (vert)

Placez 3 maisons sur la carte pour commencer.
