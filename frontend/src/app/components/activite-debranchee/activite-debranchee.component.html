Le Point de Steiner

Le point de Steiner est une position qui n'est pas forcement une ville mais qui minimise la somme des distances vers plusieurs destinations.

Pour 3 destinations, ce point forme exactement des angles de 120° avec chaque destination. C'est le Point de Fermat-Torricelli !
